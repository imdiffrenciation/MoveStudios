# MoveStudios

## Project info
MoveStudios is a community-first platform built to help creators & normal users gain more visibility, grow their fanbase, and receive support through tips. It’s designed to make creator content easily accessible to everyone with a simple design — including Web2 users — by reducing onboarding friction and improving overall reach.
**URL**: https://pin-pal-sync.lovable.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Move
